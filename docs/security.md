# Sécurité — JobPilot Secure

## Objectifs sécurité

Le projet doit montrer une approche professionnelle de la sécurité applicative.

## Fonctionnalités sécurité prévues

- Authentification sécurisée
- Hashage des mots de passe avec BCrypt
- JWT access token
- Refresh token
- Contrôle d’accès par utilisateur
- Audit logs
- Rate limiting
- Headers HTTP de sécurité
- Validation des entrées utilisateur
- Protection contre les injections SQL
- Analyse de dépendances vulnérables
- Scan SAST
- Scan Docker
- Scan DAST

## Références sécurité

Le projet suivra les bonnes pratiques inspirées de l’OWASP Top 10.
