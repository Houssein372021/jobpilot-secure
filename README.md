# JobPilot Secure

JobPilot Secure est une application web full-stack pour gérer une recherche d’emploi de manière organisée et sécurisée.

L’application permettra de gérer :

- les candidatures ;
- les entreprises ;
- les entretiens ;
- les relances ;
- les documents ;
- les statistiques ;
- la sécurité applicative.

## Objectif du projet

Créer un projet professionnel à ajouter au CV et à LinkedIn, en montrant des compétences full-stack et cybersécurité.
